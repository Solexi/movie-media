## ğŸ’» Getting Started <a name="getting-started"></a>

### Setup

Clone this repository:

```sh
  git clone https://github.com/Solexi/movie-media.git
  cd movie-media
```

### Install

Install this project with:

```sh
  npm install
```

### Usage

To run the project, execute the following command:

